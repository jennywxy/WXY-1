# Video Script Generator

This repository contains a Python utility that searches Reddit for user comments on a given topic and uses them as an introduction to build a short video script with OpenAI.

## Setup

```bash
pip install -r requirements.txt
export OPENAI_API_KEY=your_key
```

## Usage

```bash
python video_script_generator.py "your topic here"
```

The script prints a generated video script to the console.
